# Assets Directory

Place your app assets here:

- `icon.png` - App icon (1024x1024px, PNG format)
- `splash.png` - Splash screen image (1284x2778px recommended)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px, PNG format)
- `favicon.png` - Web favicon (48x48px, PNG format)

## Creating Icons

You can use online tools like:
- [App Icon Generator](https://www.appicon.co/)
- [Icon Kitchen](https://icon.kitchen/)

Or create them manually:
1. Design a 1024x1024px icon
2. Export as PNG
3. Place in this directory

## Temporary Placeholders

For development, Expo will use default icons. For production builds, you must provide proper icons.
